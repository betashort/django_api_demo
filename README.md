## midrate

* makemigrations：データベースのバージョン管理をするためのマイグレーションファイルを作成する。
* migrate：データベース構成を変更したり、その変更を取り消したりする。

```bash
python manage.py makemigrations
```

```bash
python manage.py migrate
```
